# Shop-Spectrum

</div>

### Seamlessly Explore and Shop Your Favorite Products

Shop-Spectrum is a sleek and intuitive Flutter e-commerce application that allows users to browse products from a REST API, mark favorites, and add items to their cart. With secure Firebase authentication and offline-friendly Shared Preferences, Shop-Spectrum makes shopping simple and efficient.
